# Data_mining 
Repositorio de curso minería de datos este introduce los conceptos y técnicas fundamentales para extraer conocimiento a partir de grandes volúmenes de datos. Se abordarán métodos como clasificación, clustering y reglas de asociación, junto con su aplicación en problemas reales. El objetivo es desarrollar la capacidad de analizar datos, identificar patrones relevantes y apoyar la toma de decisiones de manera fundamentada.

Se dispone para el material de las ayudantias!
